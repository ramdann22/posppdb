Jurusan RPL (Rekayasa Perangkat Lunak)

Apa itu RPL?

RPL (Rekayasa Perangkat Lunak) adalah jurusan yang mempelajari cara membuat, mengembangkan, dan mengelola perangkat lunak atau software. Di jurusan ini, kamu akan belajar bagaimana membuat aplikasi Android, website, dan sistem informasi yang digunakan di berbagai bidang.

Apa Saja yang Dipelajari?

Di RPL, siswa belajar banyak hal, di antaranya:

  • Bahasa Pemrograman (HTML, CSS, JavaScript, PHP, Python, Java)
  • Pengembangan Website dan Aplikasi
  • Database (MySQL, Firebase, dll)
  • UI/UX Design (User Interface & User Experience)
  • Testing dan Debugging Aplikasi
  • Dasar Jaringan dan Keamanan Sistem

Apa yang Dilakukan Saat Praktik?

Siswa RPL melakukan berbagai proyek nyata, seperti:

  • Membuat aplikasi kasir berbasis web
  • Merancang dan membangun website sekolah atau toko online
  • Membuat game sederhana berbasis Android
  • Mengikuti lomba coding dan IT seperti LKS atau hackathon

Prospek Kerja

Lulusan RPL memiliki peluang karir sebagai:

  • Web Developer
  • Mobile App Developer
  • Software Engineer
  • UI/UX Designer
  • Database Administrator
  • IT Support / Programmer
